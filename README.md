# 부스트캠프iOS 사전과제

## BoxOffice 앱
영화 박스오피스 앱 입니다.

## 기능
* 영화 박스오피스 리스트(리스트 형식, 격자 형식) 보기
* 영화 상세 보기
* 한줄평 등록

## 구조 및 사용 기술
* swift, urlsession
* mvc(데이터 처리 부분(model)을 viewcontroller로 부터 분리하였습니다.)
* singleton pattern(api call이 나가는 url session처리 부분을 싱글톤으로 처리하였습니다.)
* repository pattern(데이터 캐싱을 위하여 repository pattern을 적용하였습니다.)
